Método de la regla falsa

El método de la regla falsa, o método de la falsa posición, es un método iterativo de resolución numérica de ecuaciones no lineales. Este método combina el método de bisección y el método de la secante.
  1. Plantear .
  2. Establecer un intervalo inicial suponiendo que dentro de este intervalo inicial está la raíz y que la función es continua. O se puede graficar la función para establecer visualmente el intervalo inicial .
  3. A partir del intervalo inicial se calcula un punto interior .
  4. Establecer el nuevo intervalo : si , pero si .
  5. Continuar calculando c hasta que sea suficientemente pequeño.
clear; clc; clf
 
% Establecer la función f(x) como función anónima.
f = @(x) cos(0.5*x) + log10(x - 1);
 
% Visualización de la función f(x).
fplot(f) % Generar la gráfica de la función.
ax = gca; % Guardar las características del gráfico actual.
ax.XAxisLocation = 'origin'; % Centrar el eje x.
ax.YAxisLocation = 'origin'; % Centrar el eje y.
ax.Box = 'off'; % Desactivar la caja del gráfico.
ax.XTick = ax.XTick; % Mostrar las marcas numéricas del eje x.
 
% Modificación de la presentación de la gráfica.
title("Método de la regla falsa") % Añadir un título.
xlabel("x") % Añadir nombre al eje x.
ylabel("y") % Añadir nombre al eje y.
grid on % Activar la cuadrícula.
nombreFunc = func2str(f); % Convertir la función en texto.
nombreFunc = "f(x) = " + nombreFunc(5:end); % Guardar el nombre de la función.
legend(nombreFunc, "Location", "best") % Generar una leyenda.
 
% Establecer los parámetros para aplicar el método de la regla falsa.
disp("En función de la gráfica, determinar visualmente el intervalo [a, b].")
a = input("Ingresar el límite inferior del intervalo: a = "); % Solicitar 'a'.
b = input("Ingresar el límite superior del intervalo: b = "); % Solicitar 'b'.
interval = [a b]; % Guardar el intervalo [a, b].
c = (a*f(b) - b*f(a)) / (f(b) - f(a)); % Generar el primer valor de c.
tol = 1e-6; % Establecer una tolerancia.
 
% Aplicar el método de la regla falsa.
while abs(f(c)) > tol % Establecer la condición del bucle.
if f(c)*f(a) > 0 % Verificar si hay cambio de signo para actualizar el intervalo.
a = c; % Actualizar el intervalo a [c, b].
else
b = c; % Actualizar el intervalo a [a, c].
end
c = (a*f(b) - b*f(a)) / (f(b) - f(a)); % Generar el nuevo valor de c.
end
 
% Visualización de la función f(x) y su raíz en el intervalo [a, b].
fplot(f) % Generar la gráfica de la función.
hold on % Activar el mantener los ejes actuales.
plot(c, f(c), 'or', 'MarkerFaceColor', 'r') % Graficar la raíz.
hold off % Desactivar el mantener los ejes actuales.
ax = gca; % Guardar las características del gráfico actual.
ax.XAxisLocation = 'origin'; % Centrar el eje x.
ax.YAxisLocation = 'origin'; % Centrar el eje y.
ax.Box = 'off'; % Desactivar la caja del gráfico.
ax.XTick = ax.XTick; % Mostrar las marcas numéricas del eje x.
 
% Modificación de la presentación de la gráfica.
title("Método de la regla falsa") % Añadir un título.
xlabel("x") % Añadir nombre al eje x.
ylabel("y") % Añadir nombre al eje y.
grid on % Activar la cuadrícula.
legend(nombreFunc, "Raíz: x = " + c, "Location", "best") % Generar una leyenda.
 
% Impresión de resultados.
fprintf("La función %s, dentro del intervalo [%.2f, %.2f], tiene una raíz " + ...
"en x = %.4f.", nombreFunc, interval, c);
