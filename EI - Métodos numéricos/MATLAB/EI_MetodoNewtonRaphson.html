Método de Newton-Raphson

El método de Newton (conocido también como el método de Newton-Raphson) es un algoritmo para encontrar aproximaciones de los ceros o raíces de una función real. El método consiste en:
  1. Obtener la función y su derivada .
  2. Establecer un punto inicial relativamente cerca de la raíz, caso contrario es posible que el método no converja.
  3. Plantear una tolerancia adecuada.
  4. Determinar el siguiente valor mediante la expresión: .
  5. Continuar calculando valores de x hasta que , es decir, hasta que sea menor que la tolerancia establecida.

Observación

Al principio del siguiente código se utiliza la caja de herramientas matemáticas simbólicas (Symbolic Math Toolbox). Esto permitirá obtener la derivada de cualquier función que se ingrese de manera automática, sin necesidad de calcularla e ingresarla manualmente.
No obstante, luego de plantear la función simbólica y determinar su derivada simbólica , estas funciones simbólicas se convierten en funciones anónimas para trabajar únicamente con las herramientas propias de MATLAB.
clear; clc; clf
 
% Establecer la función f(x) y obtener su derivada f'(x).
syms x % Crear la variable simbólica x.
f(x) = sin(x) - cos(x^2) - 1; % Establecer la función simbólica f(x).
df(x) = diff(f(x), x); % Obtener la derivada simbólica f'(x).
 
% Convertir las funciones simbólicas en funciones anónimas.
f = matlabFunction(f(x)); % Conversión a función anónima f(x).
df = matlabFunction(df(x)); % Conversión a función anónima f'(x).
 
% Visualización de la función f(x).
fplot(f) % Generar la gráfica de la función.
ax = gca; % Guardar las características del gráfico actual.
ax.XAxisLocation = 'origin'; % Centrar el eje x.
ax.YAxisLocation = 'origin'; % Centrar el eje y.
ax.Box = 'off'; % Desactivar la caja del gráfico.
ax.XTick = ax.XTick; % Mostrar las marcas numéricas del eje x.
 
% Modificación de la presentación de la gráfica.
title("Método de Newton-Raphson") % Añadir un título.
xlabel("x") % Añadir nombre al eje x.
ylabel("y") % Añadir nombre al eje y.
grid on % Activar la cuadrícula.
nombreFunc = func2str(f); % Convertir la función en texto.
nombreFunc = "f(x) = " + nombreFunc(5:end); % Guardar el nombre de la función.
legend(nombreFunc, "Location", "southwest") % Generar una leyenda.
 
% Establecer los parámetros para aplicar el método.
x = input("Ingresar el valor inicial: x = "); % Solicitar el valor de x.
tol = 1e-5; % Establecer la tolerancia.
k = 1; % Crear un contador k para crear una medida de
% seguridad en caso de que el método no converja.
 
% Aplicar el método sucesivas ocasiones.
while abs(f(x)) > tol % Establecer la condición del bucle.
x = x - f(x)./df(x); % Calcular el siguiente valor de x.
 
% Establecer medida de seguridad.
k = k + 1; % Añadir 1 al contador k en cada iteración.
if k > 30 % Condicional de seguridad.
break % Si hay más de 30 iteraciones salir del bucle.
end
end
 
% Visualización de la función f(x) y su raíz.
fplot(f) % Generar la gráfica de la función.
hold on % Activar el mantener los ejes actuales.
plot(x, f(x), 'or', 'MarkerFaceColor', 'r') % Graficar la raíz.
hold off % Desactivar el mantener los ejes actuales.
ax = gca; % Guardar las características del gráfico actual.
ax.XAxisLocation = 'origin'; % Centrar el eje x.
ax.YAxisLocation = 'origin'; % Centrar el eje y.
ax.Box = 'off'; % Desactivar la caja del gráfico.
ax.XTick = ax.XTick; % Mostrar las marcas numéricas del eje x.
 
% Modificación de la presentación de la gráfica.
title("Método de Newton-Raphson") % Añadir un título.
xlabel("x") % Añadir nombre al eje x.
ylabel("y") % Añadir nombre al eje y.
grid on % Activar la cuadrícula.
legend(nombreFunc, "Raíz: x = " + x, "Location", "southwest") % Generar una leyenda.
 
% Impresión de resultados.
fprintf("La función tiene una raíz en x = %.4f. El método converje con %i " + ...
"iteraciones.", x, k);
