Observación

Este código solicita un número n el cual es el número de ecuaciones del sistema. A partir de n se generan la matriz de coeficientes A y el vector de términos independientes b de forma aleatoria.
clear; clc
 
% Solicitar al usuario el número de ecuaciones del sistema.
n = input("Ingresar el número de ecuaciones del sistema: ");
 
% Definir parámetros iniciales para el sistema.
limInf = -10; % Límite inferior para generar números aleatorios.
limSup = 10; % Límite superior para generar números aleatorios.
 
% Generar la matriz de coeficientes 'A' y el vector columna de términos ind. 'b'.
A = randi([limInf limSup], n); % Matriz de coeficientes generados aleatoriamente.
b = randi([limInf limSup], n, 1); % Vector de términos ind. generados aleatoriamente.
 
% Generar la matriz ampliada (A|b).
M = [A b];
 
% Realizar la eliminación de Gauss-Jordan manualmente.
for i = 1:n % Iteración sobre filas.
M(i, :) = M(i, :)/M(i, i); % Hacer 1 al pivote i-ésimo de cada fila.
for j = 1:n % Bucle para hacer 0 los elementos encima/debajo del pivote.
if i ~= j % Condicional para evitar el bucle sobre la fila i-ésima.
M(j, :) = M(j, :) - M(j, i) * M(i, :); % Hacer 0 los elementos
end % encima/debajo del pivote.
end
end
clear i j % Borrar variables i, j.
 
% Visualización de resultados.
fprintf("Las matriz de coeficientes 'A' es: \n\n"); disp(A)
fprintf("El vector columna de términos independientes 'b' es: \n\n"); disp(b)
fprintf("La matriz ampliada (A|b) es: \n\n"); disp([A b])
fprintf("La matriz reducida por filas aplicando el método de Gauss-Jordan \n" + ...
"manualmente es: \n\n"); disp(M)
fprintf("La matriz reducida por filas aplicando el método de Gauss-Jordan \n" + ...
"mediante la función interna de MATLAB 'rref' es: \n\n"); disp(rref(M))

Forma alternativa

Para iterar sobre las filas faltantes, en lugar de iterar sobre toda la fila M(i, :), se itera sobre los elementos que no son cero de la fila M(i, i:n+1).
for i = 1:n
M(i, i:n+1) = M(i, i:n+1)/M(i, i); % Forma alternativa respecto al código
% original.
for j = 1:n
if i ~= j
M(j, i:n+1) = M(j, i:n+1) - M(j, i) * M(i, i:n+1);
end
end
end
clear i j
