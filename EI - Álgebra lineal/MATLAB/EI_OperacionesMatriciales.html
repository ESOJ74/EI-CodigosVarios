Operaciones matriciales

Suma / Resta

Se define como:
O escrito de otra forma:
clear; clc
 
% Definir parámetros iniciales las para matrices que serán sumadas.
limInf = -5; % Límite inferior para generar números aleatorios.
limSup = 3; % Límite superior para generar números aleatorios.
m = 2; % Número de filas.
n = 2; % Número de columnas.
 
% Crear las matrices que serán multiplicadas.
A = randi([limInf limSup], m, n); % Matriz m x n de números aleatorios enteros.
B = randi([limInf limSup], m, n); % Matriz m x n de números aleatorios enteros.
 
% Inicializar la matriz resultante.
C = ones(m, n); % Matriz m x n de unos (donde se reemplazará el resultado de A + B).
 
% Determinar la suma matricial manualmente.
for i = 1:m % Iteración sobre filas.
for j = 1:n % Iteración sobre columnas.
C(i, j) = A(i, j) + B(i, j); % Cálculo del elemento ij-ésimo de la suma.
end
end
clear i j
 
% Visualización de resultados.
disp("Las matriz A es: "); disp(A)
disp("Las matriz B es: "); disp(B)
disp("La matriz resultante mediante el bucle es: "); disp(C)
disp("La matriz resultante mediante operaciones matriciales es: "); disp(A + B)

Producto

Se define como:
O escrito de otra forma:
clear; clc
 
% Definir parámetros iniciales para las matrices que serán multiplicadas.
limInf = -2; % Límite inferior para generar números aleatorios.
limSup = 2; % Límite superior para generar números aleatorios.
m = 2; % Número de filas de la matriz A.
n = 2; % Número de columnas de la matriz A / Número de filas de la matriz B.
p = 3; % Número de columnas de la matriz B.
 
% Crear las matrices que serán multiplicadas.
A = randi([limInf limSup], m, n); % Matriz m x n de números aleatorios enteros.
B = randi([limInf limSup], n, p); % Matriz n x p de números aleatorios enteros.
 
% Inicializar la matriz resultante.
C = ones(m, p); % Matriz m x p de unos (donde se reemplazará el resultado de A * B).
 
% Determinar el producto matricial manualmente.
for i = 1:m % Iteración sobre las filas.
for j = 1:p % Iteración sobre las columnas.
temp = 0; % Iniciar una variable temporal para el elemento ij-ésimo.
for k = 1:n % Cálculo del elemento ij-ésimo de la matriz resultante.
temp = temp + A(i, k) * B(k, j);
end
C(i, j) = temp; % Guardar la variable temporal en la posición ij-ésima.
end
end
clear i j
 
% Visualización de resultados.
disp("Las matriz A es: "); disp(A)
disp("Las matriz B es: "); disp(B)
disp("La matriz resultante mediante el bucle es: "); disp(C)
disp("La matriz resultante mediante operaciones matriciales es: "); disp(A * B)
